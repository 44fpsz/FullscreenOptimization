# 🎮 FullscreenOptimization / Optimización Pantalla Completa

Optimiza tu experiencia de juego en Windows desactivando funciones innecesarias que pueden aumentar el input lag y reducir el rendimiento en juegos competitivos.

Optimize your Windows gaming experience by disabling unnecessary system features that can increase input lag and reduce performance in competitive games.

---

## 🚀 Qué hace / What this does

- 🚫 Desactiva Xbox Game Bar / Disables Xbox Game Bar  
- 🚫 Desactiva grabaciones en segundo plano (GameDVR) / Disables background recording (GameDVR)  
- ⚡ Fuerza verdadero Fullscreen Exclusive / Forces true Fullscreen Exclusive behavior  
- 🎯 Reduce el input lag / Reduces input lag  
- 🧊 Mejora la consistencia de los frames / Improves frame time consistency  

---

## 📦 Archivos incluidos / Included Files

- **FullscreenOptimization.reg**  
  → Aplica los ajustes de rendimiento / Applies all performance tweaks for competitive gaming  

- **RevertFullscreenOptimization.reg**  
  → Restaura la configuración por defecto de Windows / Safely restores default Windows settings (undo all changes)  

---

## 🎯 Recomendado para / Recommended for

- Juegos competitivos como **Valorant, CS2, Fortnite, etc.** / Competitive games like Valorant, CS2, Fortnite, etc.  
- Jugadores que priorizan **baja latencia y rapidez de respuesta** / Players who prioritize low latency and responsiveness  
- Sistemas donde las optimizaciones de Windows generan stutter o input lag / Systems where Windows optimizations cause stutter or input delay  

---

## ⚙️ Cómo usar / How to use

1. Descarga el archivo `.reg` que necesites / Download the `.reg` file you want  
2. Haz doble clic y acepta los cambios / Double-click and accept the changes  
3. Reinicia tu PC (recomendado) / Restart your PC (recommended for full effect)  

---

## 🔄 Revertir cambios / Reverting Changes

Si quieres volver a la configuración por defecto de Windows / If you want to go back to default Windows behavior:

1. Ejecuta **RevertFullscreenOptimization.reg** / Run **RevertFullscreenOptimization.reg**  
2. Reinicia tu PC / Restart your PC  

---

## ⚠️ Notas importantes / Important Notes

- Algunas superposiciones (Discord, Xbox Game Bar) pueden dejar de funcionar / Some overlays may stop working (Discord, Xbox Game Bar)  
- Alt+Tab puede ser más lento en pantalla completa exclusiva / Alt+Tab can be slower in exclusive fullscreen mode  
- No todos los juegos se benefician por igual (DX12/Vulkan pueden tener menos impacto) / Not all games will benefit equally (DX12/Vulkan titles may see less impact)  

---

## ⭐ Consejos para mejores resultados / Tips for Best Results

- Usa **Pantalla completa** (no ventana sin bordes) / Use Fullscreen mode (not borderless)  
- Desactiva **Optimización de pantalla completa** en el `.exe` del juego / Disable Fullscreen optimizations in the game `.exe`  
- Apaga V-Sync / Turn off V-Sync  
- Activa **NVIDIA Reflex** (si tienes NVIDIA) / Enable NVIDIA Reflex (if available)
